# 提案 · 气运抽签「按愿望内容加权」

> 目标：让「气运抽签」的成功率**随用户输入的内容产生倾向**——越日常/越容易实现的愿望越容易摇高，越浮夸/越离谱/越贪心的愿望越容易摇低。
> 例：`我想吃饭` 偏高、`我想毁灭世界` 偏低。

---

## 一、可行性结论

**可行，且不需要联网/不需要 AI。**
APK 要求**纯离线**，所以不能调用大模型。但"判断一句愿望是大是小、是日常还是离谱"用一套**本地关键词词典 + 几条启发式规则**就能做到八九不离十，几十行代码、零依赖、瞬时出结果。

代价是它**不是真正理解语义**，而是"模式匹配"——词典覆盖到的词反应准，没覆盖到的词走默认（保持纯随机）。这对一个玄学小游戏来说完全够用，而且"偶尔猜不到"反而有梗。

---

## 二、三种实现路线（按推荐度）

### 路线 A：关键词词典加权（推荐 ✅）
内置一张带权重的词表：

| 类型 | 例词 | 权重 |
|------|------|------|
| 日常/温和（升） | 吃饭、喝水、睡觉、摸鱼、奶茶、早睡、散步、出门、点外卖、追剧 | +15 ~ +30 |
| 努力可达（小升） | 及格、加薪、面试、考过、减肥、表白、脱单 | +5 ~ +12 |
| 贪心/暴富（降） | 暴富、一夜、中彩票、不劳而获、白嫖、躺赢 | −20 ~ −35 |
| 浮夸/离谱（大降） | 毁灭、统一世界、称霸、长生、永生、复活、逆天改命、宇宙 | −30 ~ −45 |
| 放大词（叠乘） | 全世界、所有人、永远、一定、必须、立刻 | 把当前偏移再放大 1.3× |

**算分**：`基准 50 + Σ命中词权重`，再夹到一个均值区间（如 15~85）。

### 路线 B：启发式特征（作为 A 的补充）
不靠词、靠"句子形态"：
- **宏大度**：含「全/所有/世界/宇宙/永远」→ 降；
- **长度**：超长的"宏愿"通常更离谱 → 轻微降；
- **努力信号**：含「努力/坚持/复习/练习」→ 轻微升（天道酬勤）。

### 路线 C：本地情感词典（如知网 HowNet/大连理工情感词表）
更"正经"的中文情感极性打分。**不推荐**：体积大、对"大愿望 vs 小愿望"这种维度并不敏感（它分的是褒贬，不是难易），收益低于路线 A。

> **结论：A 为主 + B 补充。**

---

## 三、关键设计：是"偏向"，不是"写死"

为了既有倾向、又保留玄学的惊喜感，**不要直接把分数当结果**，而是：

1. 由内容算出一个**目标均值** `mean`（夹在 `15~85`）；
2. 在均值附近做**带随机的采样**（三角/近正态分布，幅度约 ±35）；
3. 最终夹到 `1~100`（建议保留极端值可能性，让"毁灭世界"也有万分之一的高光）。

效果示例（典型区间，仍可能爆冷/爆好）：

| 输入 | 目标均值 | 常见落点 |
|------|---------|---------|
| 我想吃饭 | ~80 | 68–92 |
| 我想早睡 | ~78 | 65–90 |
| 我想考试及格 | ~60 | 45–75 |
| 我想暴富 | ~22 | 8–38 |
| 我想毁灭世界 | ~12 | 3–25 |
| 我想统一宇宙 | ~8 | 1–22 |
| （空 / 没命中词） | 50（纯随机） | 1–100 均匀 |

---

## 四、注意事项 / 风险

1. **公平感**：有人就想要"纯随机"。建议加一个开关「玄学加权：开/关」（默认开）。空输入时本来就是纯随机，不受影响。
2. **可被"刷"**：用户会发现"写吃饭就高"。这其实是**乐趣点**，可当彩蛋；真要防，可对同义堆砌做衰减。
3. **词典是有限的**：没收录的词 → 走默认纯随机，不会乱给。后续可慢慢扩词。
4. **敏感内容**：词典只玩"大小/难易/贪心"维度，**不碰政治/不做道德审判**；涉及自伤等危险词，建议**不娱乐化**（直接走中性或给温和提示），避免冒犯。
5. **同日同事仍只问一次**（现有规则不变），所以同一句话不会被反复刷分。

## 五、落地成本

- 纯前端 `index.html` 内加一个 `LUCK_LEX` 词表 + 一个 `biasFor(text)` 函数 + 改一行抽签取值逻辑；
- 约 50–80 行，无新依赖，离线即用；
- 可选：加一个「玄学加权」开关（需要在界面放一个小开关，约再加 15 行）。

## 六、建议

先按**路线 A + B**做一版，默认开启、幅度温和（均值偏移 ±25~30、保留爆冷可能），上线后看手感再调词典与幅度。要不要我直接实现？要的话我会：
1. 内置一份中文词表（日常/努力/贪心/浮夸四档）；
2. 加 `biasFor()` 取目标均值 + 三角采样；
3. 加「玄学加权 开/关」开关（默认开）；
4. 空输入或未命中 → 维持纯随机，保证公平。
